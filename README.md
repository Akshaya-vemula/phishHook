# PhishHook

PhishHook is an AI-powered phishing detection system that analyzes email content to identify phishing attacks.

## Features
- Email phishing detection using Machine Learning
- Suspicious keyword analysis
- Risk score prediction
- Interactive dashboard using Streamlit

## Technologies
- Python
- Scikit-learn
- Streamlit
- Pandas

## Run the Project

pip install -r requirements.txt
streamlit run app.py
